What this demo does

Record a short 3–5 second vowel sound or phrase and the app will analyze your voice to give a risk score for Parkinson's-like voice characteristics. This is a demo project for hackathon / research; not a clinical diagnosis.

Get started — Login

Quick features

  • User sign-up & login (Firebase)
  • Record voice directly from browser
  • Upload, preprocessing, feature extraction and model inference (FastAPI backend)
  • Per-user history stored in Supabase
  • Timeline & charts per user
  • Generate PDF report and simple AI chatbot (demo)

Fill Firebase & Supabase keys in frontend/env and backend/.env as described in README before running.
